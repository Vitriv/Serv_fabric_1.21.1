# Commandes du Serveur

## Général

| Commande | Description |
| :--- | :--- |
| `/help` | Voir toutes les commandes disponibles. |
| `/msg <joueur> <message>` | Envoyer un message privé à un joueur. |
| `/w <joueur> <message>` | Alias de la commande `/msg`. |

## Téléportation (Essentials)

| Commande | Description |
| :--- | :--- |
| `/he` | Affiche l'aide des commandes Essentials. |
| `/he tpa <joueur>` | Demander à se téléporter à un joueur. |
| `/he spawn` | Se téléporter au point de spawn du serveur. |

## Parties & Claims (Open Parties and Claims)

| Commande | Description |
| :--- | :--- |
| `/openpac-parties` | Affiche l'aide des commandes d'équipe. |
| `/openpac-parties about` | Voir les informations sur les équipes existantes. |
| `/openpac-parties create <nom>` | Créer une nouvelle équipe. |
| `/openpac-parties join <équipe>` | Rejoindre une équipe existante. |
| `/openpac-parties ally` | Ajouter ou retirer des alliances entre équipes. |
| `/openpac-parties chat <message>` | Envoyer un message dans le chat de l'équipe. |
| `/opm <message>` | Alias de la commande `/openpac-parties chat`. |
| `/openpac-parties destroy` | Supprimer l'équipe actuelle. |

## Waypoints

| Commande | Description |
| :--- | :--- |
| `/waypoints` | Ouvrir l'interface de gestion des points de passage. |
